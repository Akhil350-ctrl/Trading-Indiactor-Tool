# Trading Indicator Tool 📊

This project is a simple trading analytics tool designed to help traders analyze market trends using basic indicators.

## Features
- Moving Average Calculation
- RSI Indicator
- Basic Buy/Sell Signals

## Goal
To build an open-source toolkit for traders and developers to test strategies easily.
